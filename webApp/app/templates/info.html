How to search with Blextra

Query

Put in the words you want to search for.
If you type multiple words without any operators, all terms have to appear in the texts.

For the search you can use operators.
The following operators are implemented at the moment:

& : all terms which are conctenate with these operators have to appear in the results
| : at least one of the term has to appear in the result

The operators can be used together.

Example queries:
(Sommer & Eis) | (Sommer & Bern)

Search optiones

Stemm words: If set to true, all the terms are stemmed.
Synonym expansion: If set to true, synonyms for the terms are looked up and put in the query.
(Is not implemented for french words yet)
Just search title: If set to true, the search is just done in the title of the articles.

Author

Search for a specific author.
This can be combined with "Query" and "Articles from the last ..."
or just used alone. If used alone it just gives all the articles from a this author.

Articles from the last...

Search all articles from the last ... days/weeks/months/years.

Examples

Query: Sommer Eis
Articles from the last: 1 year(s)

Looks for all articles which have the terms "Sommer" and "Eis" inside and were written in the last year.

Query: Interview
Author: Rainer Stadler

Looks for all articles which have the term "Interview" inside and were written by Rainer Stalder.
